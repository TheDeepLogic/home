> RETRO COMPUTING • VINTAGE TECH • ELECTRONICS

>> ABOUT

Welcome to my corner of the digital frontier. I'm passionate about bringing vintage computing back to life through modern tools and creative engineering. My projects blend the nostalgia of classic computers with today's technology, creating bridges between past and present.

From TRS-80 menu systems to Apple II graphics toolkits, from educational robots to physical turtle graphics implementations - each project celebrates the spirit of early computing while making these classic systems accessible and fun.

>> PROJECTS

$ TRS-80 Menu Builder

A Python script that generates a visual menu system in BASIC for TRS-80 Model III/4 computers running LDOS, MultiDOS, or LS-DOS. This tool addresses the limited availability of menu applications for TRS-80 systems by creating a fast, responsive navigation interface for your disk programs.

[VIEW PROJECT]

$ Apple II HGR Toolkit

A python script that can be used to generate HGR HPLOT commands and related AppleSoft BASIC code from an input prompt. Create stunning high-resolution graphics for your Apple II programs with the power of modern AI assistance.

[VIEW PROJECT]

$ DeLilah Bot

A custom-built educational robot created for my niece, DeLilah. This project was designed to introduce her to robotics, programming, and interactive technology in a fun and engaging way.

[VIEW PROJECT]

$ Applesoft BASIC Interpreter

A comprehensive Python implementation of the Applesoft BASIC interpreter with full graphics support using pygame. Run classic Apple II programs on modern hardware with authentic graphics and behavior.

[VIEW PROJECT]

$ RF Media Link

A universal RFID launcher system for retro computing and media. Scan RFID tags to launch emulators, open files, navigate to URLs, or execute system commands. Perfect for physical media collections (floppy disks, cartridges, VHS tapes) and retro enthusiasts.

[VIEW PROJECT]

$ Logo Turtle Bot

A physical implementation of Krell MIT Logo from the Apple II, brought to life in a 3D-printed turtle robot powered by a Raspberry Pi Pico. Experience the joy of turtle graphics in the real world!

[VIEW PROJECT]

$ RetroSerialHub

A multi-port serial hub that bridges vintage computers to modern TCP/IP networks, providing access to bulletin board systems (BBS), file transfers, and other retro computing services through a unified menu interface.
